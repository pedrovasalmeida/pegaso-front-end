# Pégaso

![Pegaso](https://i.imgur.com/B4VHNwI.gif)

Site institucional da construtora Pégaso.

---

Projeto desenvolvido utilizando algumas tecnologias como:

- **ReactJS**
- **TypeScript**
- **Context API**
- **Axios**
- **Styled Components**
- **React Router DOM** (and **History**)
